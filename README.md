## 這個Bot可以讓你不用開Minecraft就能享受掛機和跟遊戲內的人聊天，還有很多功能等你來探索
## 此Bot專門為廢土伺服器所設計

# 主要功能    
### 1.自動打怪物(支援突襲塔)
### 2.自動掛機獲得物資....
### 3.自動宣傳訊息(設施宣傳&交易頻道)  
### 4.可以不用開遊戲輸入指令&聊天  
### 5.可以透過私訊Bot來執行命令(須設定白名單)  
### 6.新增白名單指令系統(避免有心人士亂搞)  
### 7.許多自定義指令   
### 8.自動接受tpa & tpahere(需白名單)  
### 10.丟棄物品(可指定和全部丟棄)  
### 11.自動重連(防止廢土掛掉的時候無法重連)    
### 12.可直接從discord控制自己的bot(測試中功能)
